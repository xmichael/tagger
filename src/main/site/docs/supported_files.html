Tagger is built over ExifTools and can Read, Write files in the following formats:

AI AIT ARW CIFF CR2 CRW CS1 DCP DNG EPS EPS2 EPS3 EPSF ERF EXIF GIF HDP ICC ICM IIQ IND INDD INDT JNG JP2 JPEG JPG JPM JPX MEF MIE MNG MOS MPO MRW NEF NRW ORF PBM PDF PEF PGM PNG PPM PS PS2 PS3 PSB PSD RAF RAW RW2 RWL SR2 SRW THM TIF TIFF VRD WDP X3F XMP

Other formats such as MP3,AVI,PPT etc. can be Read but Writing is only supported through Custom tags and export to XMP sidecar files.
See: http://owl.phy.queensu.ca/~phil/exiftool/#supported for a full list of supported formats
